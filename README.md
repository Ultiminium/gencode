# gencode (G)

A context-dependent, density-maximized data encoding and compression system.

G defines a structured encoding alphabet where characters have types (digits, lowercase, uppercase) and adjacency rules — each character constrains what can follow it. This produces an encoding space that RNCE's arithmetic coder can exploit, delivering competitive compression with a self-describing, self-validating format.

## Pipeline

```
input → LZ77 (8MB window) → G index encoding → RNCE arithmetic coding → output
```

## Install

```bash
cargo install gencode
```

Or download a pre-built binary from [releases](https://github.com/Ultiminium/gencode/releases).

## CLI Usage

```bash
# Encode / decode (binary .g format)
g encode file.txt          # → file.g
g decode file.g            # → file

# Canonical text format (.gt) — human-readable, self-validating
g encode-text file.txt     # → file.gt
g decode-text file.gt      # → file
g validate file.gt         # checks header, charset, adjacency rules

# Stdin/stdout piping
echo "hello" | g encode-text - -o out.gt
cat out.gt | g decode-text -

# Options
g encode file.txt -n 28            # Gn instance (default: 28)
g encode file.txt --no-lz77        # skip LZ77 preprocessing
g encode file.txt --no-entropy     # skip RNCE compression
g encode file.txt --stats          # show compression stats
g info -n 28 --compare             # compare all Gn instances

# Install library + generate bindings
g install                  # installs libg to ~/.g/
g init                     # generates bindings for your project language
```

## Library Usage (Rust)

```rust
use gencode::{encode_bytes, decode_to_bytes, EncodeOptions, DecodeOptions};

let opts = EncodeOptions::default(); // G28, LZ77+RNCE enabled
let encoded = encode_bytes(b"hello world", &opts)?;
let decoded = decode_to_bytes(&encoded, &DecodeOptions::default())?;
assert_eq!(decoded.data, b"hello world");
```

### Streaming

```rust
use gencode::stream_api::{GEncoder, stream_decode};

let mut enc = GEncoder::new(28);
let mut out = Vec::new();
enc.write_header(&mut out);
enc.feed(b"first chunk", &mut out)?;
enc.feed(b"second chunk", &mut out)?;
enc.finish(&mut out)?;

let decoded = stream_decode(&out)?;
```

## Library Usage (C)

```c
#include "g.h"

// Encode
size_t cap = g_encode_max_size(in_len, 28);
uint8_t *buf = malloc(cap);
size_t out_len;
g_encode(input, in_len, 28, buf, cap, &out_len);

// Decode
size_t dec_cap = g_decode_max_size(out_len);
uint8_t *dec = malloc(dec_cap);
size_t dec_len;
g_decode(buf, out_len, dec, dec_cap, &dec_len);

// Streaming
g_encoder_t *enc = g_encoder_new(28);
g_encoder_feed(enc, chunk, chunk_len, out, out_cap, &out_len);
g_encoder_finish(enc, out, out_cap, &out_len);
g_encoder_free(enc);
```

## Library Usage (Python)

```python
import g  # generated by `g init` in your project

data = b"hello from Python"
encoded = g.encode(data)
decoded = g.decode(encoded)
assert decoded == data

# Text format
gt = g.encode_text(data)
print(gt)  # Gn/0/1/0/28/17\n...
```

## The G System

### Gn Instances

| Instance | Charset | Bits/block | Notes |
|----------|---------|------------|-------|
| G4       | 13 chars | 14 bits | smallest, highest escape rate on random data |
| G8       | 25 chars | 35 bits | |
| G16      | 49 chars | 82 bits | |
| G24      | 73 chars | 128 bits | first fully lossless instance |
| G28      | 85 chars | 128 bits | **default** — best balance |
| G32      | 97 chars | 128 bits | |
| G256     | 769 chars | 128 bits | maximum charset |

All instances G4-G256 are **fully lossless** — out-of-range indices are escape-coded transparently.

### Canonical Text Format (.gt)

```
Gn/0/1/0/28/43
00000Avf3ioXlC9gZmhvgrIcKfnN
000002itHwxJ3elOj2iSwp5nfihw
00000!7y7Npujdffz01aJbkZeuvA
```

Header: `Gn/{major}/{minor}/{patch}/{n}/{original_bytes}`  
Body: one G block per line, exactly n characters, adjacency rules enforced.  
Self-describing, self-validating, version-tagged for backwards compatibility.
